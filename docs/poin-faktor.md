# Penilaian Jabatan Berdasarkan Faktor (Metode Poin Faktor)

Untuk setiap jabatan yang ada di perusahaan, lakukan penilaian dengan langkah berikut:

1. Tentukan daftar faktor yang akan digunakan untuk menilai jabatan (contoh: Tanggung Jawab, Keahlian, Usaha, Lingkungan Kerja — atau faktor lain sesuai kebutuhan perusahaan).

2. Tentukan poin maksimal untuk masing-masing faktor. Nilai maksimal ini ditentukan bebas oleh perusahaan (tidak harus sama antar faktor, dan tidak ada aturan baku angka tertentu).

3. Untuk setiap jabatan, beri nilai poin pada masing-masing faktor (nilai poin tidak boleh melebihi poin maksimal faktor tersebut).

4. Jumlahkan seluruh poin dari semua faktor untuk mendapatkan Total Poin per jabatan.

5. Susun hasilnya dalam tabel dengan format:

| Jabatan | [Faktor 1] | [Faktor 2] | [Faktor 3] | ... | Total Poin |
|---|---|---|---|---|---|

Hasil akhir: setiap jabatan memiliki satu nilai Total Poin, yang akan digunakan sebagai input nilai jabatan (P) pada **Metode B** (Metode Dua Titik - Sistem Poin).
